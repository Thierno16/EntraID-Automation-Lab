# Entra ID Identity Lifecycle Automation

## Overview

This project demonstrates enterprise identity lifecycle automation using:

- Microsoft Graph PowerShell SDK
- Microsoft Entra ID
- PowerShell 7
- CSV-based onboarding
- Security Group automation
- Logging
- Reporting
- Git version control

## Current Features

- Graph authentication
- CSV validation
- Duplicate detection
- Enterprise logging
- User provisioning (in progress)
