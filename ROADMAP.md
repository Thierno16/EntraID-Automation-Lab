# Microsoft Entra ID Automation Lab Roadmap

## Vision

Build a complete Microsoft Entra ID Identity Lifecycle Automation Toolkit using PowerShell and Microsoft Graph.

The long-term goal is to automate common Identity and Access Management (IAM) administration tasks following enterprise best practices.
